### CountingVowels [8kyu]

**Objective** : `basic loop` `condition`

- สร้างฟังก์ชันที่ชื่อว่า getCount
- รับ parameter 1 ตัวเป็น string
- คืนค่าเป็นตัวเลขที่เท่ากับจำนวนสระที่พบใน string

Example

```js
getCount('codecamp') == 3;
```
