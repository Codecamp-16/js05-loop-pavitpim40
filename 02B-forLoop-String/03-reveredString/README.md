### RemoveSpace [8kyu]

**Objective** : `basic loop` `condition` `string`

- สร้างฟังก์ชันที่ชื่อว่า **\_\_\_** (เว้นไว้ให้คิสส)
- รับ parameter 1 ตัวเป็น string
- คืนค่าเป็น string ที่กลับหน้ากลับหลัง

Example

```js
// Input -> Output
'world'  =>  'dlrow'
'word'   =>  'drow'
```
