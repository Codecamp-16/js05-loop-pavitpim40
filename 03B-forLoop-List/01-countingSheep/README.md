### CountingSheep [8kyu]

**Objective** : ฝึกแก้ปัญหาการนับจำนวนสมาชิกใน array (counting problem)

- สร้างฟังก์ชันชื่อ countSheep
- รับparameter 1 ตัวเป็น array
- คืนค่ากลับมาเป็นจำนวนแกะ (จำนวน true)

ตัวอย่าง Input

```js
[
  true,
  true,
  true,
  false,
  true,
  true,
  true,
  true,
  true,
  false,
  true,
  false,
  true,
  false,
  false,
  true,
  true,
  true,
  true,
  true,
  false,
  false,
  true,
  true,
];
```
