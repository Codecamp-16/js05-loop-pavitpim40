### squareSum [8kyu]

- สร้างฟังก์ชันชื่อ squareSum
- รับparameter 1 ตัวเป็น array
- คืนค่าเป็นผลรวมตัวเลขของสมาชิกแต่ละตัวแบบยกกำลังสอง

ตัวอย่าง
For example, for [1, 2, 2] it should return 9 because
1^2 + 2^2 + 2^2 = 9

```js
squareSum([1, 2, 2]) == 9;
```
